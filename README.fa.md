# OpenModHeader — افزونه تغییر هدر HTTP در کروم

OpenModHeader یک افزونه سبک، متن‌باز و امن برای تغییر هدرهای درخواست و پاسخ HTTP در مرورگر Chrome است. این ابزار جایگزینی شفاف برای ModHeader است و برای توسعه وب، تست API، رفع خطاهای CORS و مدیریت هدرهای سفارشی کاربرد دارد؛ بدون ردیابی و بدون اتصال به سرور خارجی.

## امکانات

- افزودن، ویرایش، حذف یا غیرفعال‌کردن موقت هدرهای درخواست و پاسخ
- ساخت چند پروفایل و جابه‌جایی سریع بین تنظیمات مختلف هدر
- اعمال قوانین فقط روی آدرس‌های دلخواه با فیلتر URL
- توقف تمام تغییرات تنها با یک کلیک
- ورود و خروجی گرفتن از پروفایل‌های JSON سازگار با ModHeader
- ذخیره تمام تنظیمات به‌صورت محلی روی دستگاه شما

## نصب افزونه OpenModHeader در کروم

۱. برو به `chrome://extensions`
۲. **Developer mode** را روشن کن (بالا-راست)
۳. روی **Load unpacked** بزن و همین پوشه را انتخاب کن

از OpenModHeader می‌توان برای تنظیم توکن Authorization، تست API، رفع مشکلات CORS، تغییر رفتار کش و مدیریت هدرهای سفارشی HTTP در توسعه محلی استفاده کرد.

## حریم خصوصی و امنیت

این افزونه با تمرکز بر شفافیت و حفظ حریم خصوصی ساخته شده است:

- بدون سرور خارجی، بدون آنالیتیکس، بدون ردیابی.
- کد minify/مبهم نشده؛ هر خطش خواناست.
- از API استاندارد `declarativeNetRequest` استفاده می‌کند، پس افزونه اصلاً محتوای درخواست‌ها را نمی‌خواند.
- همه‌ی داده‌ها فقط در `chrome.storage.local` روی دستگاه خودت می‌ماند.
